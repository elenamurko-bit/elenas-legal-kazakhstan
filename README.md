# HR Legal AI Kazakhstan

Полностью локальная интеллектуальная система анализа трудового законодательства Республики Казахстан.

## Описание проекта

Система предназначена для анализа трудового законодательства Казахстана без использования облачных сервисов, платных API или внешних баз данных. Все данные хранятся локально, а анализ выполняется на локальной модели LLM.

## Особенности

- ✅ **Полностью локально** - без облачных сервисов и внешних API
- ✅ **Без СУБД** - все данные хранятся в обычных файлах
- ✅ **Без подписок** - никаких платных сервисов
- ✅ **AI-анализ** - локальная LLM модель для анализа законодательства
- ✅ **Комплексный поиск** - полнотекстовый, семантический, по номерам статей

## Структура проекта

```
HR-Legal-AI-Kazakhstan/
├── index.html          # Основной HTML файл
├── style.css           # Стили
├── script.js           # JavaScript логика
├── laws/               # Директория для законодательства
├── court/              # Директория для судебной практики
├── explanations/       # Директория для официальных разъяснений
├── templates/          # Директория для шаблонов документов
├── notes/              # Директория для заметок
├── settings/           # Директория для настроек
├── ai/                 # Директория для AI модели
│   ├── model.gguf      # Локальная LLM модель
│   └── llama.cpp       # Движок для запуска модели
└── update/             # Директория для обновлений
```

## Установка и запуск

### Требования

- Веб-браузер (Chrome, Firefox, Edge, Safari)
- Для AI-анализа: локальная LLM модель (например, через llama.cpp)

### Запуск

1. Откройте файл `index.html` в веб-браузере
2. Система готова к использованию

### Для полноценного AI-анализа

1. Скачайте локальную LLM модель (например, Llama 2 или аналогичную)
2. Поместите модель в директорию `ai/`
3. Интегрируйте llama.cpp или аналогичный движок
4. Настройте подключение в настройках приложения

## Модули

### 1. Законодательство

- Конституция РК
- Трудовой кодекс
- Социальный кодекс
- Законы
- Подзаконные акты
- Постановления Правительства
- Приказы министерств

### 2. Судебная практика

- Нормативные постановления Верховного Суда
- Обзоры судебной практики
- Открытые судебные решения
- Иные официально опубликованные материалы

### 3. Официальные разъяснения

- Министерство труда
- Министерство социальной защиты
- Другие официальные государственные органы

### 4. AI-анализ

Алгоритм работы:
1. Поиск в законодательстве
2. Поиск судебной практики
3. Поиск официальных разъяснений
4. Передача найденных материалов локальной LLM
5. Формирование ответа только на основании найденных документов

### 5. Поиск

Типы поиска:
- Полнотекстовый
- По словам
- По фразам
- По номеру статьи
- По номеру дела
- Семантический
- По похожим документам
